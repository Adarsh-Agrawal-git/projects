Web Development Projects (1st Year)



This repository showcases my web development projects built during my first year, focusing on core concepts of HTML, CSS, and JavaScript, along with practical implementation of logic and API integration.



Projects

1\. Tic Tac Toe



A browser-based implementation of the classic Tic Tac Toe game.



Features:



Two-player turn-based gameplay

Win detection and game state handling

Reset functionality

Clean and responsive UI



Tech Used: HTML, CSS, JavaScript



2\. Rock Paper Scissors



An interactive game where the user plays against the computer.



Features:



Randomized computer moves

Real-time result display (win/lose/draw)

Score tracking system

Dynamic UI updates



Tech Used: HTML, CSS, JavaScript



3\. Currency Converter



A real-time currency conversion tool powered by an external API.



Features:



Fetches live exchange rates using API

Converts between multiple currencies

Dynamic updates based on user input

Clean and interactive interface



Tech Used: HTML, CSS, JavaScript, API Integration



4\. To-Do List



A simple task management application for organizing daily activities.



Features:



Add and delete tasks

Real-time task updates

Persistent UI interaction

Minimal and user-friendly design



Tech Used: HTML, CSS, JavaScript
